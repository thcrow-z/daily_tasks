# Daily Tasks

Webapp simples de tarefas diarias, pronto para rodar como site estatico no GitHub Pages e ser instalado no Android como PWA.

## Publicar no GitHub Pages

1. Crie um repositorio novo no GitHub.
2. Envie estes arquivos para a branch `main`.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha `main` e a pasta `/root`.
6. Salve e aguarde o link do Pages ficar disponivel.

## Instalar no Android

1. Abra o link do GitHub Pages no Chrome do Android.
2. Toque no menu do Chrome.
3. Use `Adicionar a tela inicial` ou `Instalar app`.

O app usa armazenamento local do navegador. Para trocar de aparelho ou limpar o navegador, use a opcao de backup dentro do app antes.
